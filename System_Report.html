1. හැඳින්වීම

RDICT LMS යනු ගුරුවරුන්ට සහ සිසුන්ට අන්තර්ජාලය හරහා සහ භෞතිකව (Physical) පන්ති පැවැත්වීමට සහ අධ්‍යාපන කටයුතු කළමනාකරණය කිරීමට සාදන ලද අති නවීන Learning Management System එකකි (අධ්‍යාපන කළමනාකරණ පද්ධතියකි). මෙමගින් පන්ති ගාස්තු අය කිරීම්, වීඩියෝ පාඩම්, සජීවී (Zoom) පන්ති, නිබන්ධන (PDF) ලබා දීම සහ සිසුන්ගේ ප්‍රගතිය (Leaderboard & Marks) බැලීම ඉතාමත් පහසුවෙන් සිදු කළ හැක. පද්ධතිය සම්පූර්ණයෙන්ම ආරක්ෂිත වන අතර වීඩියෝ හොරෙන් බාගත කිරීම් වැළැක්වීමට විශේෂ ආරක්ෂිත ක්‍රමවේද භාවිතා කර ඇත.

2. භාවිතා කළ තාක්ෂණයන් (Technology Stack)

React JS & Vite
අති වේගවත් Frontend නිර්මාණය සඳහා.
Supabase (BaaS)
දත්ත ගබඩාව (Database), User Authentication සහ Payment Slips ගබඩා කිරීම සඳහා.
Google Apps Script / Drive API
නිබන්ධන (PDF) ආරක්ෂිතව ගබඩා කිරීම සඳහා.
ImgBB API
පන්ති වල Thumbnails සහ Profile පින්තූර ගබඩා කිරීම.

3. පරිපාලක (Admin) සඳහා ඇති පහසුකම්

Admin Dashboard එක මගින් මුළු පද්ධතියම පාලනය කිරීමට Admin හට සම්පූර්ණ බලය ලබා දී ඇත. Admin Panel එක ප්‍රධාන කොටස් කිහිපයකින් සමන්විත වේ:

1. Overview (සමස්ථ තොරතුරු)

මුළු සිසුන් ගණන, පන්ති ගණන, Pending Payments (අනුමත කිරීමට ඇති ගෙවීම්) සජීවීව දැකගත හැක.

[සමස්ථ පද්ධතියේ සාරාංශය පෙන්වන අලංකාර Dashboard පෙනුම]

2. Classes (ප්‍රධාන පන්ති නිර්මාණය)

නව පන්ති (උදා: 2026 ICT Theory) නිර්මාණය කිරීම. මෙහිදී පන්තියේ නම, මාසික ගාස්තුව, පන්තිය අදාළ වන Batch එක සහ ආකර්ෂණීය Thumbnail එකක් ඇතුලත් කළ හැක.

3. Sessions (පාඩම් / සජීවී පන්ති එකතු කිරීම)

ප්‍රධාන පන්තියක් යටතට අදාළ වන පරිදි සතිපතා දින වකවානු යටතේ Sessions එක් කළ හැක.

  • Recorded Videos: YouTube හි Unlisted ලෙස ඇති වීඩියෝ වල ID එක පමණක් ලබා දීමෙන් පද්ධතිය තුළට ආරක්ෂිතව වීඩියෝව සම්බන්ධ කළ හැක.
  • Live Zoom Links: සජීවී පන්ති සඳහා Zoom link එක් කිරීම.
  • PDF Uploads: Google Drive හරහා නිබන්ධන ඍජුවම සිසුන්ට බාගත කරගැනීමට හැකි වන ලෙස upload කිරීම.

4. Payment Sweeps (ගෙවීම් සහ බැංකු රිසිට්පත් අනුමත කිරීම)

සිසුන් විසින් upload කරන ලද බැංකු රිසිට්පත් (Payment Slips) පරීක්ෂා කර බැලිය හැක. නිවැරදි රිසිට්පතක් නම් අදාළ මාසය තෝරා "Approve" කළ හැක. එවිට ස්වයංක්‍රීයව එම ශිෂ්‍යයාට මාසයක් සඳහා (සහ අමතර සති 2ක සහන කාලයක්) අදාළ පන්තියට ප්‍රවේශය හිමි වේ.

5. User Management & Marks

ලියාපදිංචි වී සිටින සියලුම සිසුන්ගේ පැතිකඩ, ඔවුන්ගේ XP Points, පාසල සහ වෙනත් තොරතුරු බැලීමට සහ පරිපාලනයට Admin ට හැකියාව ඇත. එසේම සිසුන්ගේ ප්‍රශ්න පත්‍ර ලකුණු (Marks) ඇතුලත් කිරීම ද මෙතැනින් සිදු කෙරේ.

4. ශිෂ්‍ය (Student) සඳහා ඇති පහසුකම්

සිසුන්ට ඔවුන්ගේ Email සහ Password භාවිතා කර පද්ධතියට ලොග් විය හැක. Student Dashboard එක අතිශය ආකර්ෂණීය සහ පහසුවෙන් භාවිතා කළ හැකි අයුරින් නිර්මාණය කර ඇත.

1. Dashboard & Quick Stats

සිසුන්ට ඔවුන් ලබාගෙන ඇති XP ලකුණු (Gamification), ඔවුන් Leaderboard එකේ සිටින ස්ථානය සහ මෑතකදී ලබාගත් ප්‍රශ්න පත්‍ර ලකුණු ප්‍රතිඵල (Marks) Dashboard එකේ මුල් පිටුවෙන්ම දැකගත හැක.

2. Available Classes (පන්ති සඳහා ලියාපදිංචිය)

තමාගේ Batch එකට අදාළ සියලුම පන්ති මෙහි පෙන්වයි. ගාස්තු ගෙවිය යුතු පන්ති සඳහා "Enroll" බොත්තම ඔබා බැංකු රිසිට්පත (Payment Slip) Upload කළ හැක. නොමිලේ (Free) ඇති පන්ති නම් එක ක්ලික් කිරීමකින් වහාම සම්බන්ධ විය හැක.

[Payment Slip එක Upload කිරීමේ පහසු UI නිර්මාණය]

3. My Classes & Sessions (මගේ පන්ති සහ පාඩම්)

සම්බන්ධ වී ඇති සහ Admin විසින් Payment Approve කර ඇති පන්ති මෙහි පෙන්වයි. පන්තියක් ඇතුලට ගිය පසු එහි ඇති සති වාර සටහන් (Sessions) පෙන්වයි.

  • Recorded Videos නැරඹීම: වීඩියෝව සම්පූර්ණ තිරයේ (Fullscreen) නැරඹීම අනිවාර්ය කර ඇත. අතරමගදී නැවැත්වුවහොත් පසුව ආපසු එතැනින්ම නැරඹිය හැකි (Resume Support) තාක්ෂණය ඇතුලත්ය.
  • Live Classes: සජීවීව පන්ති තිබෙන දිනවලට Zoom ලින්ක් එක බොත්තමක් ලෙස පෙන්වයි.
  • PDF Downloads: Tute/PDF එකක් තිබේ නම් එක ක්ලික් එකකින් Download වේ.

4. Leaderboard & Profile

වැඩිම ලකුණු ලබා ඇති සිසුන්ගේ ශ්‍රේණිගත කිරීම් (Leaderboard) බැලිය හැක. Profile එක හරහා සිසුන්ට තම Profile Picture එක (Avatar) මෙන්ම පාසලේ නම වෙනස් කර ගත හැක.

5. පද්ධතියේ ආරක්ෂාව සහ වෙනත් විශේෂාංග (Security measures)

LMS පද්ධතිය සඳහා ඉහළම ආරක්ෂාවක් සපයා ඇති අතර Intellectual Property (බුද්ධිමය දේපල) සුරැකීම පිළිබඳව විශේෂ අවධානයක් යොමු කර ඇත.

1. Anti-Piracy Video Player (වීඩියෝ සොරකම් වැළැක්වීම)

  • වීඩියෝ පාරිභෝගිකයාගේ (Student) Email එක, නම සහ දුරකථන අංකය මෙන්ම IP ලිපිනයද වීඩියෝ එක උඩින් (Watermark) පෙන්වයි.
  • එම Watermark එක වීඩියෝව පුරා තත්පර කීපයකින් කීපයකට ස්ථානය මාරු කරයි. එමගින් Screen Record කිරීම මගින් වුවද වීඩියෝව ලබා ගැනීම අධෛර්යමත් කරයි.
  • වීඩියෝ Player එක Full Screenshot (Full Screen) මාදිලියේ පමණක් වැඩ කිරීමට සකසා ඇති හෙයින් පිටතින් වෙනත් Recording Tools ක්‍රියාත්මක කිරීම අසීරු කරයි.

2. Disable Right-Click / Developer Tools

සිසුන්ට පද්ධතිය තුළ Right-click ක්‍රියා විරහිත කර ඇත (Disabled). එසේම Keyboard Shortcuts (F12, Ctrl+Shift+I ආදිය) මගින් කේතය (Code) බැලීම වළක්වා ඇත.

3. Recording Requests (භෞතික සිසුන් සඳහා පාලනය)

Online සිසුන්ට පමණක් අදාළ පන්ති වල වාර්තාගත (Recorded) වීඩියෝ ඍජුවම පෙනෙන අතර Physical (භෞතික) සිසුන්ට ඒවා නැරඹීමට අවශ්‍ය නම් "Request Recording" හරහා Admin ට ඉල්ලීමක් යැවිය යුතුය. Admin Approve කළ පසු පමණක් වීඩියෝව Unlocked වේ. මෙය ඉතා සාර්ථක පාලන ක්‍රියාවලියකි.

4. Auto-De-activation (ස්වයංක්‍රීයව අත්හිටුවීම)

ශිෂ්‍යයෙකු ගෙවීම් කර ඇති මාසය සහ අදාළ සහන කාලය (2 Weeks Grace Period) අවසන් වූ පසු ස්වයංක්‍රීයවම පන්තිය සඳහා තිබූ ප්‍රවේශය "Expired" බවට පත් වී Video සහ Tute බැලීම අවහිර වේ. යළිත් Slip එකක් upload කර අනුමත වූ පසු පමණක් ප්‍රවේශය ලැබේ.
